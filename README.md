# Personal Site
Public personal site to display some code. Using Python FastAPI, SQLAlchemy, Tailwindcss. Deployed on DigitalOcean VPS.

---

## Setup (Linux)
```
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt

bun install
# npm install
```

---

## Scripts
```
# tailwind
./tailwind.sh watch
```
